## ✅ **Verification Queries**

After loading all data, run these queries in Supabase:

```sql
-- Total countries
SELECT COUNT(*) FROM country_crypto_profiles;
-- Expected: 50+

-- Total legislation with official links
SELECT COUNT(*) FROM crypto_legislation WHERE official_url IS NOT NULL;
-- Expected: 30-40+

-- Total physical incidents with sources
SELECT COUNT(*) FROM physical_incidents WHERE source_urls IS NOT NULL;
-- Expected: 30+

-- Total products with geography
SELECT COUNT(*) FROM products WHERE country_origin IS NOT NULL;
-- Expected: 70-100

-- Total compliance mappings
SELECT COUNT(*) FROM product_country_compliance;
-- Expected: 200+

-- Legislation by country (top 10)
SELECT
    country_name,
    COUNT(cl.id) as law_count,
    SUM(CASE WHEN cl.official_url IS NOT NULL THEN 1 ELSE 0 END) as with_official_links
FROM country_crypto_profiles ccp
LEFT JOIN crypto_legislation cl ON cl.country_code = ccp.country_code
GROUP BY country_name
ORDER BY law_count DESC
LIMIT 10;

-- Physical incidents with verified sources
SELECT
    location_country,
    COUNT(*) as incidents,
    SUM(amount_stolen_usd) FILTER (WHERE amount_stolen_usd IS NOT NULL) as total_stolen
FROM physical_incidents
WHERE verified = true
GROUP BY location_country
ORDER BY incidents DESC;
```
